# Creative Portfolios

Two individual portfolios for **Thai** (ilyrice) and **Ner**, created as part of the Web Programming Essentials discovery piscine at 42 Bangkok.

## Authors
- **Thai** (Chattumongkol Somsri) - `ilyrice`
- **Ner** - `ner`

## Project Structure
- `index.html`: Summary page highlighting both authors.
- `portfolio1.html`: Thai's personal portfolio.
- `portfolio2.html`: Ner's personal portfolio.
- `assets/`: Contains images, avatars, and certificates for the portfolios.
- `style.css`: Shared styling for the entire project.

## Features
- Responsive design using Bootstrap 5.
- Professional experience sections including internships and camps.
- Customized for each author with specific skills and backgrounds.
